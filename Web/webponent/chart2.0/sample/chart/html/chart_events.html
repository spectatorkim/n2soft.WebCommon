Chart Events

일반 디자인

beforeDrawSeriesInGraph

그래프영역별로 시리즈를 그리기 바로 직전에 발생

drawCompleted

차트 완료했을때 발생. 애니메이션 완료시점이랑 무관함.

차트 init 전에 미리 등록해야 할 경우에 jQuery selector에 bind시킬 수도 있음.
예시 : $(jQuery Selector).on('drawCompleted', function(e, chart){ ... })

주의사항

이벤트를 통하여 만들어진 SVG Node들은 reDraw, inquery 같이 다시 그려져야 하는 경우 직접 삭제해야 한다.
