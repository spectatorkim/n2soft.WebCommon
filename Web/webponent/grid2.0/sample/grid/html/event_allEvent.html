그리드 이벤트

그리드 이벤트

등락율 순위

headerRendered (Reference, Publisher)

그리드 헤더가 렌더링 됬을때 발생하는 이벤트

beforeRenderData (Reference, Publisher)

데이터를 렌더링(그리드 바디 렌더링)하기 전에 발생하는 이벤트

dataRendered (Reference, Publisher)

그리드 바디에 데이터가 렌더링 됬을때 발생하는 이벤트
하나의 로우이든 여러 로우가 추가된 경우도 발생한다.

columnResized (Reference, Publisher)

그리드 컬럼의 width가 변경되었을때

rowRendered (Reference, Publisher)

그리드에서 한 Row가 렌더링 됬을떄
실제로 Append되기전에 row 노드를 생성했을때 발생한다.

beforeAppendingRow (Reference, Publisher)

그리드에 append가 되기 직전에 발생

rowAppended (Reference, Publisher)

그리드에 append가 끝났을때

beforeCachingRow (Reference, Publisher)

그리드에 row를 캐싱하기전에 발생

rowCached (Reference, Publisher)

그리드에 row를 캐싱한 후 발생

scroll (Reference, Publisher)

그리드가 스크롤 될때 발생함
